# Atlas Notes app manual
Atlas Notes offline writing uses a local encrypted notebook cache. Sync resumes automatically after reconnecting, and conflicts are reviewed in the Sync Review panel.
Atlas Notes quick capture uses Ctrl+Shift+N on desktop. On mobile, Atlas Notes quick capture uses a two-finger swipe down and saves the note to Inbox.
Atlas Notes search repair is Settings, Search, Rebuild Index. Rebuilding the index does not delete notebooks.

# ForgeBoard app manual
ForgeBoard offline work uses a pinned board snapshot, not an encrypted notebook cache. When the device reconnects, ForgeBoard asks the user to press Merge Snapshot before syncing.
ForgeBoard quick capture uses Alt+Space on desktop. On mobile, ForgeBoard quick capture uses a long press on the plus button and saves the card to Today.
ForgeBoard search repair is Workspace Tools, Diagnostics, Reindex Cards. Reindexing cards keeps attachments intact.

# Northwind Shop support guide
Northwind Shop returns are allowed within thirty days for unused items with a receipt. The customer should open Start Return and print the prepaid label.
Northwind Shop guest tracking uses the order number and email address on the Track Order page.
Northwind Shop damaged-package support requires two photos and the shipping label in the support form.

# Harbor Outfitters support guide
Harbor Outfitters returns are allowed within fourteen days for unworn items with original tags. The customer should open Request Exchange before mailing the item.
Harbor Outfitters guest tracking uses the shipment code and billing zip code on the Find Shipment page.
Harbor Outfitters damaged-package support requires a damage description and the packing slip number.
